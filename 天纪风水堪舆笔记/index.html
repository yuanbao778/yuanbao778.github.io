基本概念

风

山

山中最是藏风。

  • 风水之山的概念包括来龙，龙虎抱，官鬼，案山，朝山，砂。
  • 树木挺拔茂盛之山主武，草木低矮秀气之山主文。
  • 形似神佛的山上则宜建寺庙，不宜建阳宅（台北观音山）。
  • 歪斜、破损、折裂、塌陷、遍布石砂是山之凶象，风水不佳。

山形九星

天下山脉，归为九类。但凡为山，皆受影响。

  • 吉：贪狼，巨门，武曲，左辅，右弼。
  • 凶：破军，禄存，文曲，廉贞。
  • 贪狼（群山连绵）：贪狼主人丁大旺，且出人聪明孝顺，财帛兴旺。
  • 巨门（双峰相等且相连）：主忠长寿，聪明孝顺，出神童。君子得之官贵，又名天医，主少疾。
  • 禄存（两小山在前，中间平凹，后有大山相靠）：出武官，重臣。
  • 文曲：后人以才艺惊世。
  • 廉贞：主沙发无道，用在正山则主兵权出武将，威震边疆。
  • 武曲：贵极，出将相。
  • 破军（两小山在前，中间交错，后有大山相靠）：子凶，腰伤，主横死。
  • 左辅：吉。
  • 右弼：吉。

龙

  • 以山势高低划分龙之部分。
    • 始祖：龙的山势最高远处。
    • 少祖：龙的山势较低缓处。
    • 山龙：龙的山势最低缓处。
  • 龙分生死。
    • 生龙：山势雄伟高大，连绵不断，青苍茂盛，生气蓬勃，看后神清气爽，精神振奋。
    • 死龙：山上光秃无树，到处塌陷，落石滚滚。
      • 凶龙：漫山细石砂子，飞沙走石。是人丁败绝，家畜死亡之地。
      • 瘦龙：山势瘦骨嶙峋，不夠饱满。不吉之地。

骨肉

  • 以土之比例论龙气之多寡，以土之颜色论龙气之旺衰。
    • 全是砂石，则气少。
    • 黑色和青色之土则气衰。
    • 红色之土则气过旺。
    • 黄色或黄中带白，且湿润之土，为最好。

来龙

坐山

  • 坐北朝南，即面朝南方，身后为北方。
  • 背对来龙才能完全吃到风水：来龙自北朝南，宅墓立向也应为坐北朝南。

穴

龙穴即上古时代火山喷出之口，如长白山天池，台湾日月潭。

  • 下面三个条件的即为龙结穴之处。
  • 龙穴之形，因地而异。
    • 龙穴以动物之象为最好：龟，鹰，大象，犀牛。
    • 或为凹形，凸形，钳形，窝形。
  • 案例：广西北流麒麟头龙穴。

龙虎抱

  • 背向来龙，则左山为青龙，右山为白虎。
  • 青龙主一房，白虎主二房。
    • 龙比虎大，则龙更能藏风（凹风），长房后代更旺。
    • 龙能管长房多久，看龙向前之余脉分支有多少（龙之官）。
      • 1条支脉管10年1，10条支脉管100年（3代）。

明堂

  • 好的明堂：一为平整宽广，二为水聚忌泄。
    • 水聚天心：明堂内有湖，湖中有岛，如日月潭中光华岛。
    • 田园明堂：明堂之上有稻田，青翠诱人，结实累累。
    • 九曲明堂：水自山上汇聚，蜿蜒曲折，回转入明堂（大富大贵之地，必出帝王）。
  • 不好的明堂
    • 水直进直出明堂：不聚财，破散祖业。
    • 破碎明堂：明堂中乱石无章，空缺一角，高低不定。
      • 住宅商家：临近之马路东补西缺，乱堆杂物。
    • 下陷明堂：明堂凹如窟窿，多有子孙败散之危。
    • 多路明堂：明堂内小路多而延伸不断，主家宅不宁，匪盗丛生。

水

  • 水要流动：要活水（家有死水，人丁败绝），但不能见出入水口（不见水来去，才藏风纳气）。
  • 水要清澈：最好有鱼有虾（水清甘美，必出功名）。
  • 水要有情：即曲折回转，九曲入明堂。
  • 水要无声：有声之水为悲吟水，主绝后。

砂

  • 砂朝向，向穴有情，背穴无情。
  • 砂以尖圆方为吉，歪斜破碎为凶。
    • 动物之象
      • 水口砂以威猛稳重灵活的动物象最好（狮虎豹猴牛象鬼龟）。
        • 台北盆地的水口砂形如狮子和绣球。
        • 桂林山水有象立水口处，似饮水状。
      • 以阴暗晦藏的动物象为凶（蛇虫鼠蚁狐）。
    • 头角峥嵘：明堂内砂石方正，头脚峥嵘，则是中流砥柱之象（后代出名将）。
      • 来龙山势高：名将力挽狂澜。
      • 来龙山势低：名将无力回天。
    • 肥肥圆圆：砂石肥圆，主聚大财（富星砂）。
    • 七零八落：尽是细细碎碎的小砂石则不好。

案山

  • 可用案山作为立向依据。
    • 案山为单：朝正中山顶为立向点。
    • 案山为双：朝两山之间为立向点。
  • 案山之形
    • 富星砂：圆肥敦厚，草木青翠。立向朝此，必出富商。
    • 水中山：耸立水中，头角峥嵘。立向朝此，国之重臣。
    • 笔架山：形如笔架，或高似笔。立向朝此，主发科甲。

朝山

  • 好的朝山

    • 高大工整，起伏有力，一层高过一层为最好（主一代强过一代）。
    • 形如太师椅：主出宰相。
    • 军营般布列：必生武将，威震边疆。

  • 不好的朝山

    • 朝山之后还有一小山头：站立可见，蹲下则不见，如窥视状，则此地必出盗贼。

    • 没有朝山：龙脉发力不过三代（百年）。

官鬼

  • 官山
    • 形如官帽，周围又似腰带缠绕：有官有带是正官，为一方之主（县长、一把手）。
      • 若再有笏板之象：则近天子。
    • 有官无带者，出偏官（老百姓选出来的民意代表）。
  • 鬼山
    • 鬼山形如兵旗，出武将，三数为大贵。
    • 鬼山势不可凶（破碎，塌陷，皆砂石）。
    • 鬼山被破坏，后代必遇小人作梗。

风水布局

帝王之地

山势雄壮

  • 山不够雄壮，即使风水再好，也非帝王之地。
  • 安徽潜山县，周围三十六座山（皇伞），但山势太过秀丽，只能出名伶（荧幕皇帝尊龙）。

文武双全

帝王风水必文武双全。

  • 近看来龙如龙椅正坐，远看朝山如太师椅端坐。
  • 远有朝山：山包起伏如军营（武官山）。
  • 近有案山：三山起伏如笔架（文官山）。
    • 远处若有高压电网，则电线高架形如笔，也主文。

九曲明堂

龙虎抱之水蜿蜒曲折，九曲回转入明堂。

天星四贵

  • 天星四贵处，宜建国都。中国之城，只有北京和西安有天星四贵。
  • 南京的上述四方也有来龙，但山势太过秀丽，不能长久（美人迟暮）。
  • 亥位上应紫微星，艮位上应天市星，巽位上应太微星，兑位上应少微星，此四星为最贵。
    • 祖坟若天星四贵得其一，亦主大贵。

环山城局

阴宅很难刚好葬到龙穴处（往往被占），可选在环山城局骑龙葬（出官带）。

  • 后代吃到城局风水之相：金城骨凸出（眉至额角之骨），可位列九卿。
  • 骑龙葬的前提是要环山，若在高山龙脊上下葬则危。
  • 罗经24座上的分经上已不可下葬，易出天灾人祸。
  • 案例：广东阿公抱孙穴。

三吉六秀

来龙方向为艮、巽、兑（酉）为三吉；朝向为艮、巽、兑（酉）、丙、辛、丁为六秀。

巨贪水局

巨门水局

风水若见巨门，配合来水方向，成巨门水合局，主少疾病，出神童，君子得之官贵。

贪狼水局

风水若见贪狼，配合来水方向，成贪狼水合局，主人丁大旺，聪明孝顺，财帛兴旺。

罗经基础

  • 罗经即罗盘，使用市面上常用的三合三元39层综合盘即可。
  • 诸君可自行在各大网购平台搜索，博主这个盘参考价格为120圆子。

奇针八法

罗经正中为天池，天池内指南针不能转动。若有异动，是为异象，称为奇针八法。

二十四山

罗经第五环为二十四山（每山为15度）。

  • 八卦四维（乾坤巽艮）：四个角。
  • 十二地支（子丑寅卯辰巳午未申酉戌亥）：正北起子，隔一填一。
    • 兑龙即为酉龙，主正西。
  • 八地支（甲乙丙丁庚辛壬癸）：东偏北起甲，填空。

天星四贵

穿山72龙

穿山72龙用以测来龙之吉凶，吉则立向，不吉不立。

  • 罗经第六环为穿山72龙，细分24山为72龙（1山管3龙），有吉凶之分。
  • 罗经倒数第三环为穿山72龙的吉凶，其中红点为吉，黑点或叉为凶（八干四维12龙为大凶）。

透地60龙

透地60龙是在定好墓碑立向之后，用以确认下葬后棺材之方位。

  • 罗经第十环为透地60龙，有”七三、三七、五、正”几种情况。
  • “正”为珠宝线，主子孙发财。

天地人盘

  • 地盘：地盘正针，用以格龙（勘定来龙方向，从而确立坐山方向）。
  • 人盘：人盘中针，用以消砂（布置风水中的砂之格局）。
    • 先有天地后有人，故人盘比地盘满半格。
  • 天盘：天盘缝针，用以纳水（测来去水之吉凶）。
    • 天最早，故天盘比地盘快半格。

八煞黄泉

  • 指24山中的八个正卦方位作为来龙方向时，建墓宅的立向禁忌。
  • 外盘为地盘正针，内盘为八曜煞（实际39层综合盘内没有，只能记口诀）。
  • 歌诀：坎龙坤兔震山猴，巽鸡乾马兑蛇头。艮虎离猪为曜煞，墓宅逢之一旦休。

八路黄泉

  • 立墓建房的来去水方位禁忌，为八路黄泉（明堂最好不见水来去，但常做不到）。
  • 八路黄泉的立向为8个方位之一，对应有方向不能有来去水，否则主孤儿寡母。
  • 测量方位时人须站在穴位上，背向来龙而面向水口。
  • 查盘方法：外盘为天盘缝针，内盘为黄泉煞（第三圈）。

坐山地煞

查盘方法：外盘为地盘正针（第五圈），内盘为地煞盘（第四圈）。

  • 指以24山作为坐山方向（立向）时，每山对应另一山，其山若有凶象，立向避之，不凶则立。
  • 如坐山为巽（坐巽朝乾），癸方有砂高耸，破碎歪斜，最凶宜避之；若体正峰圆，亦不忌。

坐山五行

  • 坐山方向有五行属性，特定方向有高砂，该方向也有五行属性，以五行生克论吉凶。
  • 排法（盘上没有，但方向以人盘中针来测）：以壬子癸起“火火土金土水”，依次顺排。
  • 拨砂决（“我”为坐山方向）
    • 克我煞见则祸绝，我生泄气渐飘零，我克奴砂为财帛；
    • 比和为旺丁财足，生我之星号食神，食神敏秀诞科甲。

    • 生： 木火土金水
    • 克：水火金木土

山形九星

外盘为地盘正针（第5圈），内盘为贪狼九星（第7圈）。

  • 用来测来龙遇山形九星时的吉凶。
    • 贪狼巨门武曲左辅右弼为吉。
    • 禄存文曲廉贞破军为凶。
  • 贪狼九星是将相同属性的山和星相匹配，比肩则合。
    • 如贪狼用于艮脉分明者：二者皆属木。
    • 来龙为艮，则坐于贪狼，或案山朝山龙虎抱明堂砂为贪狼皆应，吉。

天罡28宿

  • 天罡28星宿即28组星象，用以说明“七政”（日月金木水火土星）之位置。
  • 罗经倒数第2圈为天罡28宿：“星宿名 + 度数 + 余数”。罗盘所指方位，头顶即对应星象。
    • 度数：一圈为360度，数几就是几度。
    • 余数
      • 少：差一点（氐十六半）。
      • 太：超一点（亢九太）。
      • 半：超约半度（柳十三半）。
  • 天罡28宿五行属性如下（日月皆属火），用以与坐山和高砂五行相配，五行相同则为比肩，吉。
    • 坐向乾山（木），配奎木狼，见巽、艮、坤处有高砂（木），则为比肩，发科甲。

九星吉凶

外盘为地盘（第5圈），内盘为第7圈，测特定方向遇九星之山的吉凶。

  • 若特定方向遇凶山，大山则立向时避之，小砂则毁之。
  • 亦可依此在明堂特定方向造砂做局。

水之吉凶

八路黄泉

若立向为8个方位之一，站在穴位之上，背向来龙而面向水口，查看来去水，犯忌避之。

巨贪水局

风水见巨门贪狼之山为大吉，此时若在特定方向有来水，则与巨贪水合，更吉。

  • 若风水见巨贪，且有来水，则以此为立向依据。

  • 若风水无巨贪之山或无水，可在明堂内造砂造水做局。

巨门水局
  • 风水见巨门之山，特定方向又有来水，成巨门水合局，主少疾病，出神童，君子得之官贵。
  • 巨门用于巽脉分明者（来龙）。
贪狼水局
  • 风水见巨门之山，特定方向又有来水，成贪狼水合局，主人丁大旺，聪明孝顺，财帛兴旺。
  • 贪狼用于艮脉分明者（来龙）。

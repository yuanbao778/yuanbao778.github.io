github+hexo搭建博客

发表于 2022-11-28 更新于 2023-08-19 分类于 杂记 本文字数： 3.2k 阅读时长 ≈ 12 分钟
基于github和hexo免费搭建个人博客，主题选用的是Next。

环境配置

以Windows环境为例。

Github

  • 注册Github账号，需要邮箱（国内注册邮箱需要手机号）。
  • 创建个人仓库：Your repositories -> New -> Repository name 输入 github用户名.github.io -> Create repository。

Git

  • 下载安装Git，一路默认选项。
  • 桌面右键选择“Git Bash Here”打开终端，输入配置信息并生成密钥（三个回车）。
  • 复制/c/users/john/.ssh/id_rsa.pub里的内容，到github主页粘贴。
    • Settings -> SSH and GPG keys -> New SSH key -> Title随便起。
  • 切换到“.ssh”目录下，验证ssh密钥是否配置成功（中间一定要输入”yes”，直接回车过不去）。

Node.js

安装Hexo的前提。

  • 下载安装，一路默认。
  • cmd下检测安装是否成功。

Hexo

  • 安装hexo。
  • 初始化博客（会在当前路径新建文件夹blog_github，用于存放网站内容）。
  • 进入blog_github路径，cmd下输入三条命令，之后浏览器输入”localhost:4000”可在本地预览网站。

网站设计

主题下载

  • hexo官网挑选喜欢的主题，git下载至博客路径的theme文件夹（git网络不稳定，要多试几次）。
  • 本文选择的NEXT主题部署时乱码，在主题安装手册里直接下载源码（Latest Master Branch），解压后改名为next，放入theme文件夹后解决。

主题更换

修改blog_github路径下的_config.yml文件。

主题配置

Hexo基础配置

修改.\themes\_config.yml文件。

  • 修改Hexo基础信息。
  • 文章URL链接美化。

Next个性配置

主要修改.\themes\next\_config.yml文件。

网站信息

  • 选择子主题。
  • 修改页底powerd by相关信息：.\themes\next\layout\_partials\footer.njk。
  • 网站logo图片替换（路径.\public\images），替换图片像素要与之前一致（Windows自带画图工具即可实现）。
  • 关闭渐入动画特效。

版权声明

  • 显示版权信息。
  • 修改版权声明内容：D:\blog_github\themes\next\languages\zh-CN.yml。

菜单栏项

  • 修改next的_config.yml，增删菜单栏选项。
  • 创建菜单项对应文件目录。
  • 打开对应文件夹下index.md做进一步修改配置，以tags为例。

站内搜索

  • 添加站内搜索功能。
  • 修改.\next\_config.yml，启动本地搜索。

阅读统计

  • 下载统计插件。
  • 增改.\next\_config.yml。
  • 显示阅读进度。
